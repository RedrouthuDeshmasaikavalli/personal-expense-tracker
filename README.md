# Personal Expense Tracker

A full-stack Personal Expense Tracker developed using Flask and SQLite that enables secure user authentication, income and expense tracking, budget planning, and financial monitoring through a user-friendly web application.

## Overview
Managing personal finances effectively is essential for achieving financial stability. This project provides a secure and intuitive platform where users can track income, record expenses, monitor spending habits, and manage budgets efficiently.
The application is designed with authentication, session management, and database integration to deliver a seamless user experience while ensuring data security.

## Key Features

### Authentication & Security
- User Registration and Login
- Secure Password Hashing
- Session-Based Authentication
- Password Recovery Functionality

### Financial Management
- Income Tracking
- Expense Recording and Categorization
